Mac 初上手（2）：常用快捷键

Posted on 2018-03-02 Edited on 2018-11-05

今天就来说说快捷键吧。

想要把电脑用好，你就不能不掌握一些快捷键。

对于一些专用的行业软件，例如 Photoshop，Sketch 等，如何使用快捷键提高工作效率也是一门学问。

当然，本文并不会设计第三方软件，而只会讨论一些 macOS 上常用的快捷键。

一、空格（space）键

macOS 里，许多常见文件格式都可以直接用空格键快速预览（QuickLook），如视频、图片、音乐、页面、文档、PDF 等等。

拿图片举例，只要点选文件夹里某张图片，点击空格键就打开了，再点击上下左右键可切换浏览其他图片。

QuickLook 是预览（Preview）的功能。

Preview 的功能非常强大，是名副其实的瑞士军刀。

事实上，除了原生支持的格式，我们还可以安装各种 QuickLook 的增强插件，让其支持更多文件格式。

二、Command ⌘

这是 macOS 的另一个重要按键，它的地位可以类比 Windows 上的 Ctrl 键，通常会配合其他键完成操作。记住多数快捷组合键都需要用到它就行了。

复制粘贴

⌘ + C 复制（拷贝）
⌘ + X 剪切
⌘ + V 粘贴
⌘ + option + V 粘贴，原复制对象不会保留，相当于「移动」

关闭和退出

⌘ + W 关闭当前的软件窗口（软件并没有真正退出进程），相当于点了左上角的红色叉叉。
⌘ + Q 真正退出软件。
⌘ + option + esc 强制退出某个软件。通常在软件无响应时使用。

强制退出某个软件：

由于 Mac 内存管理机制还不差，很多时候你不需要彻底退出软件，只要关闭窗口就好，并不会对电脑性能和电池有太大的影响。

当然，这么做的前提是你的 Mac 至少有 8G 内存。

⌘ + Z 撤销

在编辑文档的时候相当于 Windows 下的 Ctrl+Z 键，即撤销刚才的操作。在 macOS 下则更为强大，你可以用这个快捷键撤销刚刚不小心的操作。

比如你刚刚删除了一张图片，但是你马上又后悔了，你当然可以点开废纸篓，找到刚刚删除的图片，然后放回原处。更好的方法是在删除后马上点击 ⌘+Z，这时候神奇的事情发生了，刚删除的图片又回到原来的位置！

⌘ + A 全选

相当于 Windows 的 Ctrl+A。

⌘ + delete 删除

即把文件移至废纸篓。

⌘ + S 保存

不用多解释你也懂。

⌘ + N 新建

新建你正在用的东西。

比如你在用 Safari，那 ⌘+N 就是新建一个 Safari 窗口。你在用 Photoshop，⌘+N 就是新建一个 PS 文稿。

⌘ + 空格键 切换输入法

也可以自己设置。

control + 空格键

显示或隐藏 Spotlight 搜索栏。

⌘ + tab（制表键）

切换 app 的，试一下你就明白了。

……

在系统偏好设置里，你可以将有些快捷键修改成你喜欢的，比如切换输入法，Spotlight 等。如果系统快捷键和第三方软件冲突，也是在这里修改。

更多更详细的快捷键可参阅 Apple 官网帮助：Mac 键盘快捷键

三、Touch Bar 和功能键

在一些 Apple 自己的键盘上，通常顶部会有如音量调节、亮度调节、媒体功能键和其他功能键。除此之外，还可以与 Fn 键组合，形成 F1、F2、F3 等标准功能键。

对于大多数用户来说，这部分功能键使用的频率并不高。

所以，自从 2017 年开始，苹果终于把创新的刀子动在这上头。取代这一行功能键的，是一个长条形的 OLED 显示屏。

这个部件被称为 Touch Bar（触控栏）。

Touch Bar 的功能很强大，想象空间也不少，尤其其即成的 Touch ID，确实使得 macOS 上的登陆和支付更加方便。

但因此而贵上一千块钱是否值得呢？

还是见仁见智吧。

四、快捷键小抄 CheatSheet

macOS 不同于 Windows，但也有规律可循，我们可以在使用的过程中慢慢熟悉，而且常用的也不会太多。

当然，如果你想要加快这个过程，这里有小抄呈上：CheatSheet。

装了这个软件后，只要长按住 ⌘-command 键，就可以显示当前程序的快捷键列表，就这么简单。

今天的功课就到这里。

Mac 初上手（1）：Mac、macOS 和各种 Store

Posted on 2018-02-27 Edited on 2018-11-05

首先，这一系列的文章主要是写给刚开始使用 Mac 的同学看的。

如果是已经有一定的使用 Mac 使用经验的朋友，下文可能对你帮助不大。

————————————————————————

对于大多数用户而言，现代操作系统并无绝对的优劣之分，哪怕有些需求操作系统自身无法满足，也可以使用第三方软件弥补。

不过，如果想要让你的 Mac 充分施展所长，你还是需要知道一些新名词，改变一些旧思维，忘掉一些 Windows 的使用习惯。

比如，你应该试试触控板，可能不需要安装某某杀毒、某某卫士，甚至可以经常不需要关机等等。

当你足够了解 Mac，你会发现她自有一套优美的哲学，让你忽略某些操作系统的细节，更专注做真正重要的事。

一、Mac? macOS?

苹果公司为自己的一系列电脑（包括 iMac、Mac mini、MacBook、Mac Pro 等）取的名字统称 Mac（原名是 Macintosh，一种苹果品种旭苹果 McIntosh 的变形）。

相对于 PC 所使用的 Windows，Mac 的专属操作系统叫 macOS（2012 年前称 Mac OS X，2012 年 - 2016 年称 OS X）。

早期的 macOS 版本以大型猫科动物为代号。比如 2000 年的 Cheetah（猎豹），2001 年的 Puma（美洲狮），2008 年的 Snow Leopard（雪豹），2012年的 Mountain Lion（山狮）等。

2013 年后，眼看猫科动物名称即将用尽，苹果又决定下一个十年采用加州景点名称作为系统代号名。例如 2013 年的 Mavericks（冲浪湾），2014 年的 Yosemite（优胜美地国家公园），2015 年的 El Capitan （酋长岩）……2017 年的 High Sierra（内华达高脊山脉）。

总结：

Mac = 苹果电脑；
macOS = 苹果电脑系统。

二、App Store、iTunes Store、Apple Store

App Store

macOS 上的 App Store 即是专属于 Mac 的软件商店，是购买和下载各种软件应用的苹果官方平台。

iTunes

iTunes 是一个重要乃至臃肿的软件工具。

它首先是一个播放器，可以用来播放和管理音乐，还可以帮助用户管理应用、图书、播客。

除此之外，它的另一个重要用途是帮助用户管理 iPhone，如备份、同步、恢复系统数据，管理应用、音乐、图书、播客，以及更新固件等。

iTunes Store

iTunes 里专门下载购买数字音乐、电影、电视节目等的商店。原本还可以访问 iOS 的 App Store，但是当前版本已经去掉了。

Apple Store 零售店

由苹果官方亲自设计、运营的官方商店，体验、购买苹果产品的地方。

Apple Store 零售店装修都很具特色（设计感）。线上的叫「Apple Store 在线商店」，线下的叫「Apple Store 零售店」，目前国内有四十几家，这个数字还在增多，具体可以到官网查找。

总结：

App Store：苹果电脑软件商店；
iTunes：在 Mac 上管理 iPhone 的工具；
iTunes Store：购买软件和数字产品；
Apple Store：苹果的官网和实体零售店。

最后

今天只说了些最基础的概念，什么是 Mac、什么事 macOS，还有各种 Store……你理清楚了么？

让我们下一篇继续。

macOS 与 Windows 的字体渲染

Posted on 2018-02-09 Edited on 2018-11-05

假如你同时用过 macOS 和 Windows，可能会发现一个很微妙但是又确实存在的区别：

这两个系统上的字体看起来不太一样。

如果是第一次看到 macOS，你可能会觉得字体看起来会有些毛茸茸、“发虚”的感觉（在 Retina 屏幕下这一点已经大幅度改善），但似乎更好看。

而一旦你适应了 macOS 的字体风格，再回头看 Windows，就会感觉那上面的字体虽然清晰，却显得有些单调丑陋。

为什么呢？

因为 macOS 和 Windows 采用不同的字体渲染策略。

对于如何在屏幕上呈现字体，苹果和微软有着不同的看法。

  • 苹果认为，字体算法的目标是尽可能还原字体的原始设计，即使代价是造成些许模糊。
  • 微软认为，字体的形状要适应像素的限制，保证清晰，容易易辨识，即使代价是字体的形状因此背离原始设计。

对于字体渲染风格的喜好或许因人而异，只不过我恰好比较喜欢苹果的风格。

事实上，这正是我第一次接触到苹果产品时最深刻的印象。

那个时候，高分辨率的屏幕还没有大行其道，而我也还是个用着神船笔记本的穷学生，一次偶然装了 Safari for Windows，却惊讶地发现，同样一个网页，在 Safari 里看起来就是比较美！

感谢那些向 Windows 世界展示了苹果美学的工程师，尽管他们后来放弃维护这个产品。

虽然那是一种毛茸茸的美，但却一下子击中了我，使我义无反顾地成为了忠实的苹果用户。

事实上，随着软硬件的不断进化，不同系统之间的字体渲染技术差距正在减少，如果有区别，也主要是各自的理念和偏好造成的。

尽管如此，macOS 的字体渲染风格在美感上还是得到较多认同的，所以诸如 GDI++、MacType 之类在 Windows 上模仿 macOS 字体渲染风格的软件层出不穷，而苹果用户则少有想要获得 Windows 字体渲染效果的需求。

从 Launchpad 到 Spotlight

Posted on 2018-02-04 Edited on 2018-11-05

注意：“启动台”和“聚焦”……都是 High Sierra 10.13.2 更彻底的中文化后才有的名字，但我还是更习惯说 Launchpad、Spotlight。

当我思考 macOS 和 Windows 有什么区别时，可能软件的管理和启动是最显而易见的。

之前的文章已经谈过软件安装包 pkg 和 dmg 的区别、软件管理工具 Homebrew，今天，来说一个更细节的问题——软件的启动。

用手机是在用软件，用电脑其实也是在用软件，虽然现在有些人在手机上基本只留在微信了，但幸好在电脑上还不至于这么寡头化吧？

闲话少说，切入正题。

方便的启动台：Launchpad

在 Windows 上，我们一般把常用软件的快捷方式放在桌面上，需要的时候从桌面打开，少数不常用的，则从开始菜单打开。再高端一点的玩法呢，经过一些设置，你还可以用 win + r 用命令的方式快速启动软件。

而在 macOS 上，虽然也可以在桌面上创建软件的替身（快捷方式），但我们一般却不这么做。

因为 Launchpad（启动台）已经足够方便。

顾名思义，Launchpad 会呈现所有 /Applications 目录下的软件，在上面你可以启动软件、创建文件夹，乃至卸载软件，如果你的软件是从 Mac App Store 上下载的话。

值得一提的是，Launchpad 的操作逻辑很像 iOS 的桌面。

你可以从 Dock（程序坞）进入 Launchpad，也可以从快捷键，又或者设置 Hot Corners（触发角）。

对于普通用户而言，停留在这里，没什么问题。

强大的聚焦：Spotlight

但是，macOS 还有一个祖传的强大武器：Spotlight（聚焦）。

Geek 不喜欢一切需要离开键盘的操作，所以——

只要按下 Command + Space（空格键），你将会打开一个新世界。

作为一个系统级别的搜索工具，Spotlight 高效而便捷，你会发现使用它启动软件实在方便不过，并且十分自然，只需要输入几个字母，出现结果后按下回车，搞定！

一切是如此的行云流水。

而且，Spotlight 不仅仅能启动软件，通过它，你还可以搜索和预览文档、计算、单位换算、搜索地理位置、系统偏好设置、播放音乐等等。

而且随着系统的更新，它还将源源不断地加入新的功能，哪怕你用不上。

Spotlight 类似上面提到过的 win + r，但是它无需额外的设置，并且更加强大得多。

如果你从来没有用过 Spotlight，那你绝对应该试试它！

也许它会使你的 macOS 更加可爱一点。- -

同样的，对于绝大多数用户而言，停留在这里，也已经足够了。

快速启动工具：Quicksilver、Alfred 和 LaunchBar

是的，人们总是如此的欲壑难填。

尽管有了 Spotlight，但还是有一些 Geek 追求着更高、更快、更强！

于是就有了 Quicksilver、Alfred 和 LaunchBar 这一类第三方启动工具。

而且，经过多年的发展，它们不但将基础的快速启动和搜索功能做到了极致，还不断地横向拓展，例如输入增强、添加日程、密码管理、剪贴板管理……甚至你还能制作和使用各种丰富的动作脚本，功能之强大复杂，以瑞士军刀譬喻都嫌力度轻了。

这三个软件都历史悠长，发展至今各有千秋，各有拥趸，如果要一一介绍起来，每个软件都不是一篇两篇文章能搞定的。

这里就不多说了。

假如你喜欢 Spotlight，又不满足它的功能，那就来试试它们吧。

在舒适和尝试之间游弋

作为普通用户，我们没有兴趣也不必要一味地追求效率和逼格，很多时候，功能只要够用，能让我们完成工作，就好了。

不过，有时候只要你往前走一步，也许就会发现新的天地。

当然，反过来也是如此，我也曾追求各种强大知名并且价格不菲（重要）的软件工具，但最后的结局往往是吃灰。

关键还是看需求。

那么，你是不是又知道了一点呢？

不需要身体接触的特雷门琴

Posted on 2018-01-30 Edited on 2018-10-01

先来假装没看标题，猜猜这些人在干啥？

……是的，他们并不是在抽风，而是在演奏！

今儿来介绍一种灰常神奇的乐器，特雷门琴（Theremin），一种利用电子阀门或真空管装置所产生声音的电子乐器。

1919 年时，俄国发明家萊昂·特雷门（Léon Theremin）发明了它，连爱因斯坦都曾参观，至今依然是世上唯一不需要身体接触的电子乐器。

1929 年约瑟·舒尔凌格为特雷门创作了一首《特雷门与乐队组曲》，并由发明家亲自演奏。

现在，听一首特雷门琴演奏的曲子感受一下。

原理

特雷门琴包含有两个像天线的突出构造，一个是长直金属杆，一个是环状的水平金属圈， 垂直杆控制音频率高低，水平环状圈控制音量大小。

其原理是利用演奏者的手与天线构成电容器，天线接在一个带有放大电路和扬声器的谐振电路上。

通过天线接受手的位置变化，改变电容的大小，从而影响其振荡回路的振荡频率来发声。

事实上，由于音色特殊，早期许多电影的恐怖气氛都是用特雷门琴来营造的。

不过，因为手的位置没有明显的记号可参考，完全靠演奏者的感觉，不容易控制音准，演奏起来有相当的难度，因此，极少有人专精此项乐器。

特雷门琴的发展与现状

特雷门琴曾被广泛应用于 40 至 50 年代好莱坞电影配乐中， “沙滩男孩”（Beach Boys） 在 60 年代采用于其流行曲中，使其更大放异彩。

日本电影《交响情人梦·完结篇》中，住在公寓阁楼的神秘作曲系女生也演奏过此乐器。

2014 年湖南卫视我是歌手第二季半决赛的竞演中，音乐总监梁翘柏老师以及周笔畅也曾使用特雷门琴进行演奏。

当然，最为人所知的，估计是在美剧《生活大爆炸（The Big Bang Theory）》（S04E12）中，Sheldon 的表演。
